# Typography

## Variables

### Font family

Xiwa uses **Poppins** as the main (sans serif) font family.

```scss
$font-family-sans-serif: 'Poppins', 'Helvetica', 'Arial', sans-serif;
```

### Font sizes

SASS variables:

```scss
$font-size-60: 11px;
$font-size-70: 12px;
$font-size-80: 13px;
$font-size-90: 14px;
$font-size-100: 16px;
$font-size-200: 18px;
$font-size-300: 21px;
$font-size-400: 24px;
$font-size-500: 32px;
$font-size-600: 36px;
$font-size-700: 48px;
$font-size-800: 56px;
$font-size-900: 64px;
$font-size-1000: 72px;

```
Example:

<Typeset

    12,

  ]}

  fontFamily="Poppins"
/>


### Font weights

SASS variables:

```scss
$font-weight-80: 200;
$font-weight-90: 300;
$font-weight-100: 400;
$font-weight-200: 600;
$font-weight-300: 800;
```
