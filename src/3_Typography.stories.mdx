# Typography

## Sass variables

### Font families

```scss
$font-family-sans-serif: 'Poppins', 'Helvetica', 'Arial', sans-serif;
```

### Font sizes

```scss
$font-size-60: 11px;
$font-size-70: 12px;
$font-size-80: 13px;
$font-size-90: 14px;
$font-size-100: 16px; // default
$font-size-200: 18px;
$font-size-300: 21px;
$font-size-400: 24px;
$font-size-500: 32px;
$font-size-600: 36px;
$font-size-700: 48px;
$font-size-800: 56px;
$font-size-900: 64px;
$font-size-1000: 72px;

```

### Font weights

```scss
$font-weight-80: 200;
$font-weight-90: 300;
$font-weight-100: 400; // default
$font-weight-200: 600;
$font-weight-300: 800;
```

### Line heights

```scss
$line-height-60: 1.1;
$line-height-70: 1.3125;
$line-height-80: 1.375;
$line-height-90: 1.4375;
$line-height-100: 1.5; // default
$line-height-200: 1.5625;
$line-height-300: 1.625;
$line-height-400: 1.6875;
$line-height-500: 1.75;
```
